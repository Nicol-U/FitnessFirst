# FitnessFirst

-- TEST 1 2 33
Inf 124 app project focused around creating a fitness app.

## Might need to install:

react
mui materials

### 1. Backend

cd server
npm install

Place a `.env` file inside `server/`:

npm run dev  
Server runs on `http://localhost:3001`.

### 2. Frontend

cd ff-app
npm install
npm start

App runs on `http://localhost:3000`.
